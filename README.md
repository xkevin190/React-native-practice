## Test App 

To compile the project you must follow these steps:

1) install dependencies

```c
yarn install

```

2) to compile on android

```c
  yarn android
  
```

3) to compile on ios
```
  cd ios && pod install && cd ..
  
  yarn ios
  
```

4) to run unit tests 

```
  yarn test
```
